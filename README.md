# Homebrew Tap for prompt-cheater

Homebrew tap for [prompt-cheater](https://github.com/KIMGYUDONG/prompt-cheater) - Convert natural language to Claude-friendly XML prompts.

## Installation

```bash
brew tap KIMGYUDONG/prompt-cheater
brew install prompt-cheater
```

## Usage

After installation, set your Gemini API key:

```bash
export GEMINI_API_KEY="your_api_key_here"
```

Then run:

```bash
cheater
```

## Requirements

- macOS or Linux
- Tmux
- Python 3.10+
